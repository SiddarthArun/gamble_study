# Gambling themed pomodoro chrome extension.

## Features
- Start a logged study session with the start session button. This tracks your pomodoro sessions as well as your gambling outcomes
- Hit "Spin to win" to spin for a pomodoro session in slots mode or switch to blackjack mode and play for a time
- Add tasks with the tasks button
- Block distractions with the block button - Just type "youtube" to block youtube.com and so forth
- If you like ambient noise, there is a small button in the top right to enable that

## Installation Instructions
- Download the repository as a .zip above
- Extract the files into a new folder
- In chrome, go to chrome://extensions/
  - In the top right, enable developer mode
  - Click on "Load Unpacked" in the top left, and select the new, extracted folder
  - Finally, press the update button
